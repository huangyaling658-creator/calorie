## 安全架构

API 密钥已全部从前端代码迁移到云函数：

| 调用场景 | 方案 |
|---------|------|
| 食物图片识别 | 前端 → 云函数 `aiProxy` → DashScope |
| 文本食物分析 | 同上 |
| 运动图片识别 | 同上 |
| 运动文本分析 | 同上 |
| 语音实时转写 | 微信同声传译插件（本地，无需密钥） |
| 数据同步 | 前端 → 云函数 `dataSync` → 云数据库 |

## 本地开发步骤

1. 打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. **导入项目**，目录选 **`热量记录`** 文件夹（不是 `miniprogram` 子文件夹）
3. AppID 使用 `project.config.json` 中配置的 `wxdd9242e27c46445c`
4. **开通云开发**：
   - 点击开发者工具左上角「云开发」按钮
   - 创建云开发环境（免费额度即可）
   - 记住环境 ID（当前为 `cloud1-5gnlocol78e0ddcc`）
5. **创建云数据库集合**：
   - 在云开发控制台 → 数据库 → 新建集合 `user_data`
   - 权限设为「仅创建者可读写」
6. **部署云函数**：
   - 右键 `aiProxy` → 「上传并部署：云端安装依赖」
   - 右键 `dataSync` → 「上传并部署：云端安装依赖」
   - 在云开发控制台将 `aiProxy` 超时时间设为 **60 秒**
7. 编译运行，即可在模拟器或真机预览中使用
